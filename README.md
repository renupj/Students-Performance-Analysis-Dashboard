# 🎓 Student Performance Analysis Dashboard

## 📌 Project Overview
This project is a Power BI dashboard designed to analyze student performance data and provide insights into academic achievements, attendance patterns, and subject-wise results.

The dashboard helps educators and institutions monitor student progress, identify performance trends, and make data-driven decisions.

## 🚀 Features

- Student performance overview
- Subject-wise marks analysis
- Attendance tracking
- Grade distribution visualization
- Top and low-performing student identification
- Interactive filters and slicers
- Dynamic KPI cards

## 📊 Dashboard Insights

- Total Students
- Average Marks
- Attendance Percentage
- Subject Performance Comparison
- Grade Distribution
- Student-wise Performance Analysis

## 🛠️ Tools & Technologies

- Power BI Desktop
- Microsoft Excel / CSV Dataset
- Data Cleaning & Transformation
- Data Visualization
